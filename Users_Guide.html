PowerServer User's Guide
 
Command Usage
Commands are issued in the following format.
/rcon command "[commandname] [arguments]"
For instance, to burn player number 6, you would issue the following command in the console.
/rcon command "burn 6"
You can issue multiple commands by separating them with a semi-colin.
/rcon command "burn 6;mortar 12"
Many commands can also be performed on multiple players. To force players 5 and 9 to join spectator you'd issue the following command.
/rcon command "force spectator 5 9"
Commands without arguments do not require opening or closing quotation marks. For instance, the command endmap, does not have any arguments.
/rcon command endmap
 
Punishment Commands
PowerServer features several different methods to wreak havoc on llamas, cheaters, team stackers and other undesirables via player punishment commands. In all cases, the player entity number is used to target a specific COD player. You can view the player numbers by typing /rcon status in the console.
The burn command lights a player on fire, killing them. Example usage:
rcon command "burn 7"
The explode command blows up a player, killing them. Example usage:
/rcon command "explode 16"
The mortar command shoots a series of mortars at a player's position until he has died. Example usage:
/rcon command "mortar 5"
The cow command turns a player into a flaming cow, burning them until they are dead. Example usage:
/rcon command "cow 3"
The disarm command causes a player to drop all of his weapons, including both weapon slots, grenades and pistol. Example usage:
/rcon command "disarm 14"
 
Map Voting
When enabled, map voting occurs at the end of each map after the scoreboard has been displayed. Players are presented with three choices randomly selected from the map vote rotation. In addition to this a gametype can be assigned to each map candidate. During map voting the gametype assigned to the map candidate is displayed to the right of the map. If there is no assigned gametype for a map candidate, then whatever gametype the server is currently running is used.
Players vote by pressing their fire button until the indicator (the star icon) is positioned next to their desired choice. Their latest selection is printed in the bottom left corner of the screen.
The admin has the capability of overruling the players' votes and forcing a particular map candidate to win via the forcevote command.
/rcon command "forcevote [map number]"
The map number is either the number 1, 2 or 3 and represents the first, second and third map choices displayed respectively. Example usage:
/rcon command "forcevote 2"
The above command would force the second displayed map candidate to win.
 
Match Modes
PowerServer's Match Mode features makes the administration of competitive matches unbelievably simple. Match Mode has three different internal modes, PreMatch, Count Down and Live. To place the server into PreMatch simply use the matchsetup command. Example usage.
/rcon command matchsetup
During PreMatch the HUD displays the status of important cvars such as timelimit, scorelimit, roundlength, freelook, teamspec and others. If the map should end during PreMatch because the timelimit or scorelimit is hit, the server will restart the map, keep all players on their current teams and place the server back into PreMatch. This prevents the map from ending and the server switching to another map while players are still getting prepared. To prevent players from killing each other and interrupting team or clan communications all players are made invulnerable to damage during PreMatch.
PowerServer comes pre-configured with nearly every league and ladder config available for Call of Duty. The PreMatch HUD displays which, if any, match config is currently loaded. The following are all available commands that will load the appropriate league or ladder config:
/rcon command cal
/rcon command twl_tdm_ladder
/rcon command twl_tdm_league
/rcon command twl_sd_ladder
/rcon command twl_sd_league
/rcon command twl_re_ladder
/rcon command twl_hq_ladder
/rcon command cb_tdm_ladder
/rcon command cb_hq_ladder
/rcon command cb_re_ladder
/rcon command cb_sd_ladder
/rcon command ugs
When you execute any of the above commands several things happens. FIrst, PowerServer loads the appropriate game and server settings for each config. Secondly, if the current gametype does not match the loaded config, it changes the gametype and reloads the map. And lastly, PowerServer disables all other CoDaM plugins, to prevent any other plugins from interfering with league gameplay. PowerServer's own Punishment Commands, Random Mortars, Map Voting, Spawn Protection and Mercy Limit are also disabled during MatchMode. You can issue any of the config commands any number of times during PreMatch mode.
One of the most powerful features of MatchMode is the Ready Up button. During PreMatch players can toggle their Ready Up status by hitting their USE button. The PreMatch HUD displays how many players are currently ready on each team. In addition, every ten seconds the server prints a list of players who are not Ready.
Once all players are Readied Up, the server will automatically start the Count Down. In this manner, once a server is placed into PreMatch mode a match can be ran virtually unattended. Admins can also force the match to start with the matchstart command. Example usage:
/rcon command matchstart
The Count Down displays on the HUD and lasts for fifteen seconds. When the Count Down hits zero, the server restarts the map, keeps all players on the same team and puts the server into Live mode. Inside Live mode, PowerServer is virtually unnoticeable aside from a small message in the corner of the HUD displaying "MatchMode" and if the teams are locked or unlocked. PowerServer's ability to lock teams means that players can connect and spectate the match without being able to interfere in the game by joining up.
When the match ends, PowerServer displays the scoreboard twice as long as normal, allowing all players ample time to check and record scores and to get screenshots. After that, the map is restarted, players are kept on the same teams and the server is placed back into Match Setup mode.
During Live mode the matchsetup command can be issued anytime to place the server back into Match Setup. Match Mode can be shut off at anytime by using the matchstop command. Example usage:
/rcon command matchstop
 
Team & Player Control
PowerServer includes a variety of methods for controlling which teams players are assigned to and can join. The teamlock command can prevent any player from changing teams, including spectators. When the teams are locked and a player attempts to switch teams their menu is closed and a message is displayed telling them that the teams are currently locked. Example usage:
/rcon command teamlock
It can also be used to prevent individual players from changing teams. Example usage:
/rcon command "teamlock 7"
The command teamunlock can be used in the same format to unlock the teams or to unlock individual players. Example usage:
/rcon command teamunlock
/rcon command "teamunlock 7"
PowerServer also allows you to force individual players to a different team with the force command. Example usage:
/rcon command "force axis 3"
/rcon command "force allies 12"
/rcon command "force spectator 6"
You can even use the force onmultiple players with one command. Example usage:
/rcon command "force allies 5 7 13 20"
By using the forceallspec command, all players on either team can be forced to become spectators. Example usage:
/rcon command forceallspec
Admins also have the ability to assign a specific clantag to either axis or allies using the axisclan and alliedclan commands. When a clan assignment is issued, PowerServer will check all player names and force anyone with the designated clan tag to join the appropriate team. In addition to this, if a player with a designated clantag attempts to join a team they will be autoswitched to whatever team their clantag was assigned to. Players with designated clantags are allowed to go spectator whenever they like. Example usage:
/rcon command "axisclan [BY0B]"
/rcon command "alliedclan -=S0bz=-"
A clantag assignment can be cleared at any time be issuing the axisclan or alliedclan commands without arguments. Example usage:
/rcon command axisclan
/rcon command alliedclan
Clantag assignments can also be swapped with the swapclans command. Example usage:
/rcon command swapclans
Any players who do not have an assigned clantag, can be made to join a specfic team using the noclan command. Example usage:
/rcon command "noclan axis"
/rcon command "noclan allies"
/rcon command "noclan spectator"
 
Miscellaneous Commands
You can print a line of bold text across all players HUD using the say command. Example usage:
/rcon command "say d00dz this game r0x0rz j00 b0x0rz!!!"
To end a map immediately and display the scoreboard use the endmap command. Please note, that this command is disabled in MatchMode. Example usage:
/rcon command endmap
For admins who like to have a little fun PowerServer includes a fun little command called matrix. Once issued the "matrix mode" lasts for approximately ten seconds. The matrix command cannot be explained. It must be seen. Example usage:
/rcon command matrix
